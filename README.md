# 幸运大转盘

一个简单实用的在线抽奖转盘工具，支持手动输入项目数量或导入.txt文件自定义选项。

## 功能特性

- **动态转盘生成**：根据设定的项目数量动态创建彩色扇形
- **文件导入支持**：支持导入.txt文件自定义抽奖选项
- **流畅动画效果**：4秒平滑旋转动画，带有自定义缓动函数
- **灵活配置**：可随时调整项目数量或重新导入文件
- **直观界面**：清晰的指针指示，实时显示中奖结果

## 使用方法

### 方法一：输入项目数量
1. 在"项目数量"输入框中输入数字（建议1-60之间）
2. 点击"更新转盘"按钮生成转盘
3. 点击"开始抽奖"进行抽奖

### 方法二：导入项目名单
1. 准备一个.txt文件，每行写一个项目名称
2. 点击"导入项目名单(.txt)"按钮选择文件
3. 文件加载完成后，点击"开始抽奖"进行抽奖

## 文件格式要求

导入的.txt文件需满足以下要求：
- 每行一个项目名称
- 支持中文、英文及特殊字符
- 空行将被自动忽略

示例文件内容：
```
一等奖
二等奖
三等奖
谢谢参与
再来一次
```

## 技术特性

- 纯HTML/CSS/JavaScript实现，无需依赖
- 响应式设计，适配不同屏幕尺寸
- 使用CSS conic-gradient创建扇形效果
- 平滑的CSS3旋转动画
- 三角函数计算文本旋转角度，确保文字正向显示

## 浏览器兼容性

支持所有现代浏览器（Chrome, Firefox, Safari, Edge等）
